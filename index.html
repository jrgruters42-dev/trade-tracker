📋 Caruso-Aligned Harvest Strategy

Rule 1: Locked in YTD Protection
IF Locked in YTD < $0
THEN harvest enough to make it $0 or positive
Rule 2: Unrealized Risk Management
IF Unrealized Risk > 70%
THEN harvest 25-50% of largest winner
Rule 3: Path to Breakeven Progress
WHEN a winner is up >3R
THEN harvest 25% to chip away at Path to Breakeven
Rule 4: Never Give Back a Full Winner
WHEN a position hits +5R
THEN lock in at least 50% (move stop to +2.5R minimum OR sell 50%)

🔴 The Round-Trip Problem You're Solving

The Painful Cycle:
Week 1: Lost $500 on 3 small trades (Realized: -$500)
Week 2: GLW is up $3,000 unrealized → YTD looks great! (+$2,500)
Week 3: GLW pulls back to +$1,000 → YTD now +$500
Week 4: GLW stops out at breakeven → YTD: -$500

Result: You made $3,000, gave it ALL back, ended negative.

The Fix: When GLW was up $3,000 in Week 2, harvest 25-50% to lock in profit and cover those losses. Then if it pulls back, you're still protected!

One-Time Setup (5 minutes)

Follow these steps to enable automatic cloud sync:

  1. Go to firebase.google.com
  2. Click "Get Started" → "Add Project"
  3. Name it "Trade Tracker" → Continue
  4. Disable Google Analytics → Create Project
  5. Click "Realtime Database" → Create Database → Start in test mode
  6. Click ⚙️ Settings → Project Settings
  7. Scroll down → Click `` (web icon)
  8. Register app: "Trade Tracker"
  9. Copy the config values and paste below:

ℹ️ This config is saved in your browser. You only need to do this once per computer.

Troubleshooting "Connecting to cloud..."

  1. Make sure Database URL starts with https:// and ends with .firebaseio.com
  2. In Firebase Console, go to Realtime Database → Click Rules tab
  3. Make sure rules are set to test mode: 
    {
  "rules": {
    ".read": true,
    ".write": true
  }
}
  4. Click Publish in Firebase Console after changing rules
